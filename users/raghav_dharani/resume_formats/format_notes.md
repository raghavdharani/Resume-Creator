# Raghav Dharani Resume Format Notes

Preferred current format:

- Restrained navy tone
- Centered name, title, and contact header
- Light blue section bands with navy rule
- Strong but ATS-safe section hierarchy
- Clean hyphen bullets for PDF compatibility
- Tight bullet alignment and spacing
- Two-page senior product resume format
- Avoid decorative icons, complex tables, columns, or nonstandard glyph bullets

Known issue:

- Some PDF viewers rendered symbol bullets as question marks. Use hyphen bullets for final PDFs unless a specific viewer-safe bullet has been verified.
